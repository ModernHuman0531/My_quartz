# G-code
* 指令集

| 指令       | 功能            |
| -------- | ------------- |
| G0       | 快速定位          |
| G1       | 直線切削          |
| G2       | 順時針切削         |
| G3       | 逆時針切削         |
| G4       | 暫停            |
| G20      | 英制單位          |
| G21      | 公制單位          |
| G90      | 絕對座標          |
| G91      | 相對座標          |
| M03      | 雷射開啟          |
| M05      | 雷射關閉          |
| M18      | 程式結束          |
| M05 S0   | 用於確實關閉先前加工的雷射 |
| G1 F600  | 設定雷射進給率       |
| G4 P0    | 暫停指令(準備開啟雷射)  |
| M03 S255 | 開啟雷射 並設定功率    |
|          |               |
* 常用指令使用方法
1. (G0/G1) X10. Y12.: 如果是絕對座標系，代表從**原點**走到座標(10,12)，如果是相對座標系，則代表從上一個點走一個向量(10,12)
2. (G2/G3) X10. Y12. I10. J6.: X,Y是代表終點的座標，相對座標系化則是代表終點鄉對於起點的座標，而I,J則是代表起點指向原心的向量，I代表X軸，J代表Y軸
* 常用模板
```g code
M05 S0
G91
G21
G1 F600
G0 X0. Y0.// start point
G4 P0
G1 F600
M03 S255
G4 P0
G1 F600
...
G4 P0
M05 S0
G0 X0. Y0.
G4 P0
M03 S255
G4 P0
G1 F600
// 若是多圖形的話要暫停移到起始位置，設定完再開始
G4 P0
M05 S0
G0 X0. Y0.
M18
//結束模板

```
