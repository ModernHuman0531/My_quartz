# Ranger
* Vum-like file manager
## Download and Setup
1. Download in arch linux
	`sudo pacman -Sy ranger`
2. Copy the original setup configurations
	* `ranger --copy-config=all` , This will generate three files in `~/.config/ranger` folder.
	* rc.conf (For key binding and UI setup)
	* rifle.conf (Open file's rules)
	* scope.sh (previewing usage script)`
3. The default text editor is nano if you press `enter`, if you want to use vscode, press `r` and enter `code`.
