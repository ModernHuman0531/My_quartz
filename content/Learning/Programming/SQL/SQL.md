# SQL
* SQL(Structured query Language) is a declarative language, that help us deal with the Database like table or else.
* Declarative language (SQL):
	*  The program is a **description of the result**.
	* Interpreter is find out the method of generating that result.
* Imperative language(python and scheme):
	* The program is the **description of computing process**.
	* The interpreter is to show the rule of execution.
