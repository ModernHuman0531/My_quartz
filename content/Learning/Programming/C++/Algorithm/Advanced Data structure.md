## 樹狀數組(BIT)
當有問題需要有兩個operations
1. 更新pos位置的數值為x
2. 查詢`[1,i]`區間的區間和(前綴和)
時，儘管可以用線段樹來實現，但在`[1,n]`區間實現線段樹所需要花的空間為**4n**，但在BIT裡可以用空間**n**來實現同樣的操作，因此我們偏向用BIT來實做要更動數值的前綴和。
* BIT定義:
BIT每一個位置儲存的值是根據位置的編號來決定，BIT_i的定義是以A_i為結尾，儲存長度為lowbit(i)的區間值，BIT_i=`[i-lowbit(i)+1,i]`

**lowbit(i)是將i拆解成2進位，將除來最小位數的1以外數值都變0後再轉成10進位。**
舉i=56為例，56=111000,lowbit(56)=1000=8,BIT_56=`[56-8+1,56]`=`[49,56]`，代表儲存A矩陣裡第49到第56位置,在程式上以`i&-i`求得
* BIT的性質
BIT編號是從1開始，0只是一個空間不代表任何區間
* 節點i的父節點為**i-lowbit(i)**
* 節點i的深度與i二進制的1數量相同
* 節點i的右兄弟節點為**i+lowbit(i)**
節點0就是跟節點，根據上述性質建樹後，得到的區間有以下性質
1. i節點的父節點是p，p=i-lowbit(i)，帶入區間公式後，發現**節點i的區間為`[p+1,i]`**
2. 節點i與他的左兄弟節點父親都相同，但節點i的終點比他們更右邊，因此節點i的區間包含他左兄弟的區間
3. 節點i的左兄弟j的子孫節點區間也包含在i區間裡，因為子孫區間一定介於`[j,i-1]`之間，一定被i節點區間包含

### 單點修改
* 如果現在要更新pos位置的值，等同要更新包含所有pos位置的區間，包含了
1. pos位置所有的右兄弟節點 **(pos+lowbit(pos))**，區間性質2
2. pos位置的父節點的所有右兄弟節點 ，假設pos最右邊區間的兄弟節點為t，其父節點的右兄弟節點為 **t+lowbit(t)**
這代表我們只要在超出數量範圍前一直更新pos+=lowbit(pos)就能更新完所有包含pos的區間，此動作的複雜度為log(n)
```c++
void modify(int pos,int x){
	for(;pos<=n;pos+=lowbit()){
		更新pos的值
	}
}
```
### 前綴查詢
要找到以pos為結尾的區間，要是很多不重疊的區間，且區間的聯集要恰好是我們想求的區間和。
區間以pos為結尾的區間恰好是pos，且此區間緊接在他的父節點後，他的父節點是pos-lowbit(pos)，代表只要我們一直往父節點上找直到根節點前，將這些和相加便可得到區間為`[1,pos]`的前綴和
```c++
int query(int pos){
	ans=0;
	for(;pos>0;pos-=lowbit(pos)){
		ans+=a[pos];
	}
	return ans;
}
```
