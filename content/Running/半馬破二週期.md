# 半馬破二週期
三月跟四月我已經培養起基礎的跑量了，月跑量都有破100km，因此現在制定接下來的兩個月課表，第一個小目標是5km破25min，而再來就是最終目標是半馬破二，之前都是以半馬破二為目標訓練，我以為照著週期走5km就可以順利破25min，但今天嘗試破二時發現到第三公里在505心肺功能就不給力了加速不上去，因此我把一個小目標訂到5km破25min，下一個月在將長距離加上去
## 小目標一：5km破25min
一週目標菜單：間歇跑、節奏跑、長距離跑、zone1-zone2輕鬆恢復跑

週一到週日
Easy run -> 健身/Easy run -> Tempo run -> Easy run -> 間歇跑 -> 休息 -> Long run
### 每週結構
間歇：(Rest 3min)
* week 1:
	* 1km * 4 @ 5:00
* week 2
	* 1km * 5 @ 5:00
* week 3
	* 1km * 4 @ 4:50
* week 4
	* 1km * 5 @ 4:50
Tempo:
* week 1:
	* 5km @ 5:25-5:15
* week 2:
	* 5km @ 5:25-5:15
* week 3:
	* 5km @ 5:20-5:10
* week 4;
	* 5km @ 5:20-5:10
Long run:
分為兩種：一種是純zone 2，一種是zone 2混zone 3，隔週使用
* 10-12km 純Zone2
* 10-12km Zone2-Zone3: 前70%在zone2，後30%在zone3，一定不能超過zone4!!!
Easy run:
* 45-60min Zone 1-Zone 2，最好是zone 2偏慢一些，大概是zone 1-zone 2交界
